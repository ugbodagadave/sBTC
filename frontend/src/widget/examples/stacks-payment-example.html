sBTCPay Stacks Payment Integration

Overview

This example demonstrates how to integrate sBTC payments with the Stacks blockchain using the sBTCPay widget. The widget provides seamless integration with Stacks-compatible wallets for direct blockchain transactions.

How it works:
  1. User connects their Stacks wallet (e.g., Leather Wallet)
  2. Payment details are sent to the sBTCPay backend
  3. A transaction is created on the Stacks blockchain
  4. The widget monitors transaction confirmation
  5. Payment is marked as complete in the sBTCPay system

Key Features

  • Stacks Wallet Integration - Connect directly with Stacks-compatible wallets
  • Smart Contract Payments - Payments are processed through the payment-processor smart contract
  • Real-time Transaction Monitoring - Track payment status on the blockchain
  • Fallback Options - QR code payment for non-Stacks wallets
  • Responsive Design - Works on desktop and mobile devices

Basic Implementation

To integrate the sBTCPay widget with Stacks payments, include the widget script and initialize it with your payment intent:

<!DOCTYPE html>
<html>
<head>
    <title>sBTCPay Stacks Payment</title>
</head>
<body>
    <!-- Container for the payment widget -->
    <div id="sbtcpay-widget"></div>

    <!-- Include the widget script -->
    <script src="https://unpkg.com/@sbtcpay/widget/dist/widget.min.js"></script>
    
    <script>
        // Initialize the widget
        document.addEventListener('DOMContentLoaded', function() {
            new SBTCPayWidget({
                apiKey: 'your_api_key_here',
                paymentIntentId: 'your_payment_intent_id',
                container: '#sbtcpay-widget',
                theme: {
                    primaryColor: '#3498db',
                    backgroundColor: '#ffffff'
                },
                onSuccess: function(paymentData) {
                    console.log('Payment successful!', paymentData);
                    alert('Payment completed successfully!');
                },
                onFailure: function(error) {
                    console.error('Payment failed!', error);
                    alert('Payment failed: ' + error.message);
                }
            });
        });
    </script>
</body>
</html>
